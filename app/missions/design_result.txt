Voici la charte graphique définie pour le service de drop-servicing B2B professionnel :

**Palette de couleurs**

* Bleu marine : `#4567b7` (utilisé pour les éléments principaux et les boutons de navigation)
* Blanc : `#ffffff` (utilisé pour les éléments de fond et les zones de texte)
* Or : `#ffd700` (utilisé pour les boutons d'appel à l'action et les éléments de mise en avant)
* Orange : `#ffa07a` (utilisé pour les éléments de mise en avant et les alertes)

**Typographie**

* Inter (police principale)
* Montserrat (police de secours)

**Hiérarchie**

* Boutons d'appel à l'action (CTA) à haut contraste :
 + Couleur de fond : `bg-orange-500` (`#ffa07a`)
 + Couleur de texte : `text-white` (`#ffffff`)
 + Taille des boutons : `text-lg` ou supérieure
* Zones de texte principale :
 + Police : `font-sans` (Inter)
 + Taille : `text-2xl` ou supérieure
 + Couleur de texte : `text-gray-900` (`#1a1d23`)
* Zones de texte secondaire :
 + Police : `font-sans` (Inter)
 + Taille : `text-lg`
 + Couleur de texte : `text-gray-600` (`#333333`)

**Ambiance**

* Épurée : utiliser les marges et les paddings pour créer une espace entre les éléments
* Rassurante : utiliser des éléments visuels pour créer une atmosphère de confiance
* Haut de gamme : utiliser des éléments visuels pour créer une atmosphère de luxe et de sophistication
* Orientée conversion : utiliser des éléments visuels pour inciter à l'action

/* Zones de texte principale */
<h1
